## 3. インターフェース窓口一覧

### 3.1 監査法人とのインターフェース

| 項目 | 内容 |
|------|------|
| 担当パートナー | （指名） |
| 主担当者 | （指名） |
| 定例会議 | 月1回 / 第2週金曜日 |
| 主な協議事項 | ・新システムでの内部統制評価方針<br>・勘定科目体系の変更影響<br>・収益認識タイミングの妥当性<br>・電子証憑の監査手続き<br>・為替再評価・有価証券評価の処理確認 |
| 連絡方法 | メール、必要時電話会議 |

**【注意点】**
- システム変更に伴う会計処理変更は事前協議必須
- J-SOX評価対象範囲の変更は早期に報告
- 電帳法対応の証憑保管方法は監査手続きに影響
- 新収益認識基準の適用はパートナーレベルで確認

| 項目 | 内容 |

- 軽減税率・非課税判定ロジックの確認
