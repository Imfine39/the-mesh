## 4. 仕入税額控除ルール

### 4.1 控除要件

| 要件 | 内容 | 例外 |
|------|------|------|
| 適格請求書の保存 | 原本またはスキャン保存 | 3万円未満の公共交通機関等 |
| 帳簿への記載 | 所定の事項を記載 | - |
| 取引の課税性 | 課税仕入に該当 | 非課税取引は控除不可 |

### 4.2 帳簿記載要件

| 記載事項 | 説明 | 仕訳データ対応 |
|----------|------|----------------|
| 課税仕入れの相手方の氏名・名称 | 仕入先名 | vendor.name |
| 取引年月日 | 仕入日 | ap_invoice.invoice_date |
| 取引内容 | 商品・サービス名 | ap_invoice_line.description |
| 対価の額 | 税込金額 | ap_invoice.total_amount + total_tax |

### 4.3 仕入税額控除計算

| 区分 | 計算方法 | 適用条件 |
|------|----------|----------|
| 全額控除 | 課税仕入税額の全額 | 課税売上割合95%以上かつ課税売上5億円以下 |
| 個別対応方式 | 用途別に控除計算 | 課税売上割合95%未満 |
| 一括比例配分方式 | 一律に比例配分 | 課税売上割合95%未満（選択適用） |

### 4.4 個別対応方式の計算

```
控除税額 = 課税売上対応分の全額
         + 共通対応分 × 課税売上割合

課税売上割合 = 課税売上高（税抜）÷ 総売上高（税抜）
```

### 4.5 課税売上割合の計算

| 分子 | 分母 |
|------|------|
| 課税売上高（税抜） + 輸出免税売上高 | 課税売上高（税抜） + 輸出免税売上高 + 非課税売上高 |

## 5. 端数処理ルール

### 5.1 消費税計算の端数処理

| 処理方法 | 説明 | 適用 |
|----------|------|------|
| 切り捨て | 1円未満を切り捨て | **当社採用** |
| 切り上げ | 1円未満を切り上げ | 選択可 |
| 四捨五入 | 1円未満を四捨五入 | 選択可 |

### 5.2 インボイス制度での端数処理要件

**【重要】**
- 税率ごとに1回の端数処理（明細ごとではない）
- 1つのインボイスにつき税率ごとに1回のみ

```
【計算例】
請求書内訳：
  商品A（10%）: 1,234円
  商品B（10%）: 2,345円
  商品C（8%）:    567円
─────────────────────────
10%対象合計: 3,579円 → 消費税: 357.9円 → 357円（切捨）
 8%対象合計:   567円 → 消費税:  45.36円 → 45円（切捨）

合計税抜: 4,146円
合計消費税: 402円
合計税込: 4,548円
```

### 5.3 明細計算との差額調整

| 調整方法 | 内容 |
|----------|------|
| 最終明細調整 | 最後の明細行で差額を調整 |
| 別行追加 | 調整行を別途追加 |
| 当社採用 | 最終明細調整方式 |

## 6. 税コード判定ロジック

```
[売上取引発生]
      ↓
[取引先が国内か？]
├─ No → [輸出先か？]
│        ├─ Yes → S00（免税）
│        └─ No → STA（対象外）
└─ Yes → [取引内容は？]
         ├─ [コンサル/IT/事業用賃貸]
         │    └─ S10（課税10%）
         ├─ [飲食料品/新聞]
         │    └─ S08（軽減8%）
         ├─ [住宅賃貸/有価証券売却/利息]
         │    └─ SHI（非課税）
         └─ [配当金/保険金]
              └─ STA（不課税）
```

### 6.2 仕入時の税コード判定フロー

```
[仕入取引発生]
      ↓
[取引先が国内か？]
├─ No → [輸入取引か？]
│        ├─ Yes → IMP10/IMP08
│        └─ No → PTA（対象外）
└─ Yes → [課税仕入か？]
         ├─ No → [非課税か？]
         │        ├─ Yes → PHI（非課税）
         │        └─ No → PTA（対象外）
         └─ Yes → [インボイス登録事業者か？]
                  ├─ Yes → P10/P08（全額控除）
                  └─ No → [経過措置期間内か？]
```

### 8.3 5%ルールへの対応

有価証券売却が多い場合、課税売上割合計算への影響を考慮。

| 有価証券売却額 | 非課税売上に算入（×5%） |
| 課税売上割合への影響 | 売却額の5%のみ分母に算入 |
| 仕入税額控除 | 割合が95%を下回る場合は個別対応方式 |
