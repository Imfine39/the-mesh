## 2. 締め処理順序

### 2.1 サブレジャ→GL連携順序

```
[締め処理フロー]

Level 1（並行処理可能）─────────────────────
├─ [AR締め] 売掛金残高確定
│     └─ T&M請求、固定額請求、賃料請求
├─ [AP締め] 買掛金残高確定
│     └─ 外注費、経費精算、その他費用
├─ [FA締め] 資産残高・償却確定
│     └─ 減価償却実行、資産移動反映
└─ [経費締め] 経費仕訳確定
      └─ 承認済経費の計上

Level 2（Level 1完了後）─────────────────────
├─ [為替換算] 外貨残高再評価
│     └─ 外貨建債権債務の期末換算
├─ [有価証券評価] 時価評価
│     └─ 投資有価証券の時価評価
└─ [前払/繰延振替] 期間按分処理
      └─ 前払費用、前受収益の振替

Level 3（Level 2完了後）─────────────────────
└─ [GL締め] 総勘定元帳締め
      └─ 貸借一致確認、残高確定

Level 4（GL締め完了後）─────────────────────
├─ [連結パッケージ] 連結用データ作成
├─ [税務申告データ] 申告用データ作成
└─ [経営報告] 月次報告資料作成
```

### 2.2 締め処理チェックリスト

| 処理 | チェック項目 | 確認方法 | 許容差異 | 担当 |
|------|--------------|----------|----------|------|
| AR締め | 売掛金残高と明細一致 | 残高照合レポート | ゼロ | 経理 |
| AR締め | 入金消込完了 | 未消込残高一覧 | ゼロ | 経理 |
| AP締め | 買掛金残高と明細一致 | 残高照合レポート | ゼロ | 経理 |
| AP締め | 支払消込完了 | 未消込残高一覧 | ゼロ | 経理 |
| FA締め | 資産台帳とGL一致 | 資産照合レポート | ゼロ | 経理 |
| FA締め | 償却計算完了 | 償却明細 | - | 経理 |
| SEC締め | ポートフォリオとGL一致 | 有価証券照合レポート | ゼロ | 財務 |
| 為替締め | 為替レート適用確認 | レート一覧 | - | 経理 |
| GL締め | 貸借一致 | 試算表 | ゼロ | 経理 |
| GL締め | 前期繰越一致 | 期首残高照合 | ゼロ | 経理 |

### 2.3 締め処理ステータス管理

| ステータス | 説明 | 操作可否 |
|------------|------|----------|
| OPEN | 入力可能期間 | 仕訳入力・修正可 |
| SOFT_CLOSE | 仮締め（承認者のみ入力可） | 承認者のみ入力可 |
